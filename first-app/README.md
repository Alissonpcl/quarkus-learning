# quarkus-first-app project

Project created to learn Quarkus Framework

## Commands

| Description | Command |
| :--- | :--- |
| Running in dev mode | ```./mvnw compile quarkus:dev``` |  
| Create a native executable | ```./mvnw package -Pnative``` |
